# Team Roles and Responsibilities

## Overview

The project is divided among team members based on their skills and responsibilities. Proper role distribution helps in efficient project development.

## Team Roles

### Project Manager

Responsibilities:
- Manage overall project progress
- Track deadlines
- Coordinate team activities
- Monitor project requirements


### Machine Learning Developer

Responsibilities:
- Collect and analyze datasets
- Perform data preprocessing
- Train machine learning models
- Evaluate model performance


### Backend Developer

Responsibilities:
- Develop Flask application
- Integrate machine learning model
- Handle prediction requests
- Manage application logic


### Frontend Developer

Responsibilities:
- Design user interface
- Develop HTML pages
- Implement CSS styling
- Improve user experience


### Documentation and Testing Member

Responsibilities:
- Prepare project documents
- Create test cases
- Record testing results
- Prepare final presentation
