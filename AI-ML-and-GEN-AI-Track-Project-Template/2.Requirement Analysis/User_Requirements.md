# User Requirements

The Flood Prediction System should provide an easy platform for users to predict flood risks.

## User should be able to:

- Enter geographical and environmental information.
- Submit input data for prediction.
- View flood risk prediction results.
- Understand the predicted risk level.

## Administrator should be able to:

- Manage datasets.
- Train and update machine learning models.
- Monitor system performance.
- Improve prediction accuracy.
